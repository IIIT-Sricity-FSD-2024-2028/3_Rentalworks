## 🎨 Figma Design

The user interface and user experience of the **PG Rental Property Management System** were designed using **Figma**. The design illustrates the complete workflow of the platform, including property discovery, booking requests, tenant dashboards, complaint management, and service requests. It provides a clear visual representation of how different users such as **Guests, Tenants, Owners, Wardens, and Admins** interact with the system.

### 🔗 Figma Prototype

An interactive prototype that demonstrates the **navigation flow, user interactions, and screen transitions** within the system. The prototype helps visualize the complete user journey, starting from browsing PG properties to booking a room and managing tenant services.
